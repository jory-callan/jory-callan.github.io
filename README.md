# jory-callan.github.io

Personal site built with [Astro AntfuStyle Theme](https://github.com/lin-stephanie/astro-antfustyle-theme).

Thanks to [Stephanie Lin](https://github.com/lin-stephanie) for the theme and [Anthony Fu](https://github.com/antfu) for the design inspiration.

## Tech Stack

- [Astro](https://astro.build/) 5
- [UnoCSS](https://unocss.dev/)
- [pnpm](https://pnpm.io/)

## Development

```bash
pnpm install
pnpm dev     # local dev server
pnpm build   # build to dist/
pnpm preview # preview build
```
